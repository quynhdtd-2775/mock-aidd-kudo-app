| `DEFAULT_LOCALE ∈ SUPPORTED_LOCALES` | Default is supported |
| `LOCALE_COOKIE === 'NEXT_LOCALE'` | Cookie name correct |

✅ **Result:** All 15 tests pass. Path-traversal guard confirmed; contract security enforced.
